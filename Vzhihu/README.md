# Vzhihu - A Greasemonkey/Tampermonkey Script

为新版知乎页面添加快捷键支持



| 快捷键   | 说明         |
| ----- | ---------- |
| j / k | 上一个/下一个答案  |
| g/G   | 第一个/最后一个答案 |
| /     | 搜索         |
| o     | 显示评论       |
| v     | 赞同         |
| d     | 反对         |
| t     | 感谢         |
| sc    | 收藏         |
| fx    | 分享         |
